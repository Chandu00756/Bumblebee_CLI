# homebrew-bumblebee

Homebrew tap for [Bumblebee CLI](https://github.com/Chandu00756/Bumblebee_CLI) — a dependency security scanner for macOS.

## Install

```sh
brew tap chanduchitikam/bumblebee
brew install bumblebee-cli
```

## Usage

```sh
bee --version          # verify install
bee scan               # scan your whole machine
bee status             # show scanner status
```

## Update

```sh
brew upgrade bumblebee-cli
```
